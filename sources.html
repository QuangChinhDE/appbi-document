Kết nối dữ liệu — Data Sources

Điểm khởi đầu của mọi báo cáo: kết nối trực tiếp tới cơ sở dữ liệu, kho phân tích đám mây, bảng tính và file — dữ liệu được truy vấn live, không cần sao chép thủ công.

Nhiều loại nguồn, kết nối trong vài phút

Hỗ trợ PostgreSQL, MySQL, BigQuery, Google Sheets và Manual Table (Excel/CSV). Kết nối được tự kiểm tra khi lưu.

  • BigQuery: đăng nhập Google (OAuth) hoặc Service Account JSON.
  • Google Sheets: chọn spreadsheet/sheet, đọc/ghi dòng.
  • Manual Table: kéo-thả CSV/Excel (nhiều sheet), tự suy kiểu cột.
⚙ Cách làm
  1. Vào Data Sources → New Data Source.
  2. Chọn loại nguồn, nhập kết nối → Create (tự test).

Quản lý & chia sẻ tập trung

Danh sách mọi kết nối kèm thống kê, tìm/lọc theo loại-quyền-chủ sở hữu, và test kết nối ngay trên từng dòng.

  • Chia sẻ theo Xem / Sửa / Toàn quyền (dùng chung cho cả Dataset, Chart, Dashboard, Workboard).
  • Chặn xoá khi nguồn đang được Dataset sử dụng.
