Kết nối dữ liệu — Data Sources

Điểm khởi đầu của mọi báo cáo: kết nối trực tiếp tới cơ sở dữ liệu, kho phân tích đám mây, bảng tính và file — dữ liệu được truy vấn live, không cần sao chép thủ công.

Trong trang này

1.1Danh sách nguồn dữ liệu

Là gì: nơi quản lý mọi kết nối — thống kê, tìm kiếm, lọc theo loại/quyền/chủ sở hữu, xem dạng lưới hoặc bảng.

⚙ Cách làm

  1. Mở Data Sources ở thanh điều hướng trái.
  2. Dùng ô Search hoặc bộ lọc All source types để tìm nhanh nguồn.
  3. Trên mỗi dòng, dùng các nút Test connection, Share, Delete ở cột Actions.
Lưu ý: không xoá được nguồn đang được một Dataset sử dụng — hệ thống sẽ báo các Dataset phụ thuộc để bạn xử lý trước.

1.2Tạo nguồn mới & các loại nguồn

Là gì: AppBI hỗ trợ PostgreSQL, MySQL, BigQuery, Google Sheets và Manual Table (Excel/CSV). Kết nối được kiểm tra tự động khi lưu.

⚙ Cách làm

  1. Bấm New Data Source.
  2. Đặt Name và chọn Type (loại nguồn).
  3. Nhập cấu hình tương ứng:
    • PostgreSQL/MySQL: host, port, database, schema, user, password.
    • BigQuery: đăng nhập Google (OAuth) hoặc dán Service Account JSON.
    • Google Sheets: kết nối Google rồi chọn spreadsheet/sheet.
    • Manual Table: kéo-thả file .csv/.xlsx vào vùng tải lên.
  4. Bấm Create — hệ thống tự test kết nối; nếu lỗi sẽ báo ngay.
Mẹo: với Manual Table, file Excel nhiều sheet sẽ được nạp tất cả các sheet thành nhiều bảng.

1.3Sửa cấu hình & kiểm tra kết nối

Là gì: cập nhật thông tin kết nối an toàn. Thông tin nhạy cảm (mật khẩu, khoá) được mã hoá; để trống nghĩa là giữ nguyên giá trị cũ.

⚙ Cách làm

  1. Bấm vào tên nguồn để mở chi tiết, rồi bấm Edit.
  2. Sửa các trường cần thiết (mật khẩu để trống nếu không đổi).
  3. Bấm Update/Save — kết nối được kiểm tra lại tự động.
  4. Hoặc bấm Test connection ở danh sách để kiểm tra mà không cần sửa.

1.4Chia sẻ & phân quyền

Là gì: cấp quyền cho người dùng/nhóm với 3 mức Xem / Sửa / Toàn quyền. Cơ chế này dùng nhất quán cho cả Dataset, Chart, Dashboard, Workboard.

⚙ Cách làm

  1. Bấm biểu tượng Share trên nguồn cần chia sẻ.
  2. Nhập email người dùng (hoặc chọn nhóm), chọn vai trò Viewer/Editor/Full.
  3. Bấm Share. Có thể thu hồi quyền bất cứ lúc nào trong cùng hộp thoại.
Mẹo: cần viết SQL tuỳ ý? Bạn không cần công cụ ngoài — dùng bảng SQL Query trong Dataset hoặc chế độ Custom SQL trong Explore.
