Explore — Trình tạo biểu đồ

Biến dữ liệu thành hình ảnh qua 3 bước (Loại biểu đồ → Gắn dữ liệu → Định dạng) với khung xem trước trực tiếp. Hỗ trợ 33 loại biểu đồ trên 6 nhóm.

33 loại biểu đồ, 6 nhóm mục đích

Essentials · Comparison · Trend · Composition · Relationship · Geo — chọn nhóm rồi chọn loại phù hợp.

  • Xu hướng → Line · So sánh → Bar · Tỷ trọng → Donut · KPI → thẻ số · Bản đồ → Point/Region Map.

Bố cục 3 khu vực — nhìn là hiểu

① Thanh trên (Dataset, Run/Save, Builder↔SQL) · ② Cột cấu hình 3 bước · ③ Khung xem trước (Chart / Table / Query).

⚙ Quy trình
  1. Chọn loại → gắn trục & giá trị (kèm hàm tổng hợp).
  2. Run xem thử → chỉnh Format → Save.

Định dạng đẹp & minh bạch truy vấn

Bảng màu, định dạng số, nhãn dữ liệu, legend, Top/Bottom N. Tab Query hiện SQL chạy thẳng trên nguồn — bằng chứng dữ liệu real-time.
